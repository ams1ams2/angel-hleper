# ملخص الميزات - الملاك المساعد v2

## 🎯 نظرة عامة
مساعد ذكي متحرك على سطح المكتب يستخدم الذكاء الاصطناعي لتقديم اقتراحات ذكية ومحادثة فورية باللغة العربية.

---

## ✨ الميزات الرئيسية

### 1. 🎨 شخصية متحركة تفاعلية
- شخصية ملاك متحركة على سطح المكتب
- حالات متعددة: خمول، حديث، سعيد، مساعدة، تنبيه، نائم
- تأثيرات بصرية: هالة، شرارات، توهج، حركة سلسة
- تفاعل مع الماوس: تكبير وتغيير الحالة عند التمرير
- حركة سلسة وانتقالات متحركة

### 2. 💡 اقتراحات ذكية تلقائية
- تحليل تلقائي للشاشة كل دقيقة أو عند تغيير النافذة
- اقتراحات عملية بناءً على ما تراه على الشاشة
- تركيز على: اختصارات، ميزات مخفية، تحسينات تجربة المستخدم
- استخدام GPT-5.2 عبر OpenRouter API

### 3. 💬 محادثة فورية
- **F12**: فتح/إغلاق نافذة المحادثة من أي مكان (حتى خارج التطبيق)
- محادثة مباشرة بالعربية
- ردود مختصرة وواضحة (3-5 جمل)
- ذاكرة المحادثة (آخر 16 رسالة)

### 4. 🧠 ذاكرة ذكية
- حفظ تفضيلات المستخدم وأنماط الاستخدام
- تتبع التطبيقات المفضلة
- تخصيص الاقتراحات حسب احتياجاتك
- تجنب تكرار نفس الاقتراحات
- حفظ تاريخ الاقتراحات (آخر 240 اقتراح)

### 5. ⌨️ اختصارات لوحة المفاتيح
- **F12**: فتح/إغلاق نافذة المحادثة (يعمل عالميًا من أي نافذة)
- **F11**: تحريك الشخصية لموقع عشوائي على الشاشة

### 6. 🎨 واجهة مستخدم جميلة
- تصميم عصري بألوان بنفسجية وذهبية
- فقاعات كلام ديناميكية
- تأثيرات بصرية سلسة
- واجهة PyQt6 احترافية

---

## 🔧 التقنيات المستخدمة

- **PyQt6**: الواجهة الرسومية
- **OpenRouter API**: GPT-52 للذكاء الاصطناعي
- **mss**: التقاط الشاشة
- **Pillow**: معالجة الصور
- **Threading**: معالجة متزامنة للمهام

---

## 📋 المتطلبات

- Windows 10/11
- Python 3.8+
- مفتاح OpenRouter API (مجاني)

---

## 🚀 الاستخدام السريع

1. **تشغيل البرنامج**: `python main_new.py`
2. **الاقتراحات**: تظهر تلقائيًا كل دقيقة
3. **المحادثة**: اضغط **F12** من أي مكان واكتب سؤالك
4. **تحريك الشخصية**: اضغط **F11** لتحريكها
