## 📖 نظرة عامة

الملاك المساعد هو تطبيق سطح مكتب مبتكر يجمع بين:
- **🎨 شخصية كرتونية متحركة** تطفو على سطح المكتب
- **🤖 ذكاء اصطناعي متقدم** باستخدام GPT-5.2 عبر OpenRouter
- **💡 اقتراحات ذكية تلقائية** بناءً على ما تفعله على شاشتك
- **💬 محادثة فورية** عبر F12 من أي مكان
- **🧠 ذاكرة ممتدة** تتعلم من تفضيلاتك واستخدامك

---

## ✨ المميزات الرئيسية

### 🎯 الاقتراحات الذكية التلقائية
- مراقبة ذكية للشاشة وتحليل النوافذ النشطة
- اقتراحات مخصصة حسب البرامج المستخدمة
- تنبيهات أمان وخصوصية
- اختصارات لوحة مفاتيح مفيدة
- نصائح لتحسين الإنتاجية

### 💬 محادثة فورية عالمية (F12)
- اضغط **F12** من **أي نافذة** لفتح الشات
- تركيز تلقائي فوري على مربع النص
- ردود ذكية سياقية
- محادثة طبيعية بالعربية

### 🎨 شخصية متحركة جذابة
- رسومات احترافية بتقنية OpenGL
- حركات طبيعية (تنفس، رفرفة أجنحة، تحليق)
- فقاعات كلام جميلة ومريحة للعين
- تأثيرات بصرية (هالة، شرارات، توهج)
- حالات متعددة (مساعدة، تنبيه، سعيد، نائم)

### 🧠 ذاكرة ذكية
- تحفظ تفضيلاتك وأنماط استخدامك
- تتعلم من التطبيقات المفضلة لديك
- تخصص الاقتراحات حسب احتياجاتك
- تتجنب تكرار نفس الاقتراحات

## 🚀 التثبيت والإعداد

### المتطلبات الأساسية

- **Windows 10/11** (للميزات الكاملة)
- **Python 3.8+**
- **مفتاح OpenRouter API** (مجاني - [احصل عليه هنا](https://openrouter.ai/keys))

### خطوات التثبيت

#### 1️⃣ استنساخ المشروع
```bash
git clone https://github.com/yourusername/angel-assistant.git
cd angel-assistant
```

#### 2️⃣ إنشاء بيئة افتراضية (موصى به)
```bash
python -m venv venv
venv\Scripts\activate
```

#### 3️⃣ تثبيت المكتبات المطلوبة
```bash
pip install -r requirements.txt
```

#### 4️⃣ إعداد ملف البيئة
```bash
# انسخ ملف المثال
copy env.example .env

# افتح .env وأضف مفتاح OpenRouter API الخاص بك
# OPENROUTER_API_KEY=your_api_key_here
```

#### 5️⃣ تشغيل البرنامج
```bash
python main.py
```

## 🎮 كيفية الاستخدام

### الاقتراحات التلقائية
- البرنامج يعمل تلقائياً في الخلفية
- سيظهر اقتراح كل دقيقة أو عند تغيير النافذة
- الشخصية تظهر في زاوية الشاشة وتعرض الاقتراح في فقاعة كلام

### المحادثة الفورية
1. اضغط **F12** من **أي مكان** (حتى لو كنت في برنامج آخر)
2. اكتب سؤالك أو طلبك
3. اضغط **Enter** أو زر "إرسال"
4. انتظر الرد الذكي من المساعد

### التفاعل مع الشخصية
- **تحريك الماوس** فوق الشخصية: تكبير وتغيير حالة
- **السحب**: يمكنك تحريك الشخصية (إذا فعّلت هذه الميزة)
- **التحرك التلقائي**: الشخصية تتحرك بشكل عشوائي كل فترة

## ⚙️ التخصيص

### تعديل الإعدادات الأساسية
افتح `main.py` وعدّل المتغيرات في القسم `CONFIG`:

```python
# حجم الشخصية
CHARACTER_SIZE_W = 320
CHARACTER_SIZE_H = 420

# مدة الانتظار للاقتراحات
CAPTURE_EVERY_SECONDS = 60

# الشاشة المستخدمة (1 = الأساسية)
MONITOR_INDEX = 1
```

### تغيير الألوان
```python
COLORS = {
    "primary": "#7C3AED",   # اللون الأساسي
    "accent":  "#F59E0B",   # لون التمييز
    "alert":   "#EF4444",   # لون التنبيهات
    "help":    "#22C55E",   # لون المساعدة
}
```

### تخصيص البرومبت (Prompt)
عدّل `SYSTEM_PROMPT_VISION` و `SYSTEM_PROMPT_CHAT` لتغيير أسلوب الردود.

## 🔧 حل المشاكل الشائعة

### ❌ F12 لا يعمل من نوافذ أخرى
- تأكد من تثبيت `pywin32`: `pip install pywin32`
- قد تحتاج تشغيل البرنامج كمسؤول (Run as Administrator)
- بعض البرامج تحجب المفاتيح العالمية (مثل الألعاب بوضع ملء الشاشة)

### ❌ خطأ في الاتصال بـ API
- تأكد من صحة مفتاح OpenRouter API في ملف `.env`
- تحقق من اتصال الإنترنت
- راجع رصيد حسابك على OpenRouter

### ❌ الشخصية لا تظهر
- تأكد من دقة الشاشة وإعدادات العرض
- جرب تغيير `MONITOR_INDEX` إذا كان لديك أكثر من شاشة
- تأكد من عدم وجود برنامج آخر يحجب النوافذ العلوية

### ❌ البرنامج بطيء
- قلل `CAPTURE_EVERY_SECONDS` لتقليل عدد الاقتراحات
- تأكد من سرعة اتصال الإنترنت (API يعتمد على السحابة)
